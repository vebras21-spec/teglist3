Файлы с расширенем .html открываются в браузере.

  • Назначение текста
  • Выделение текста
  • Графические элементы
  • Структура страницы

Справочник тегов.

1. Теги для текста

  • p - абзац от англ. paragraph
  • h1..h6 - заголовки от англ. heading
  • ul/ol - маркированный и нумерованный списки от англ. unordered/ordered list li - пункт списка от англ. list item

2. Теги для выделения текста

  • b - жирный от англ. bold
  • i - курсивный от англ. italic
  • del - перечеркнутый от англ. delete

3. Теги для графических элементов

  • hr - линия от англ. horizontal rule
  • img - картинка от англ. image

4. Теги задающие структуру страницы

  • !DOCTYPE html - не тег а инструкция для браузера
  • html - заключает в себя всё содержимое веб-страницы
  • head - для хранения служебной информации о странице
  • body - содержимое документа html
  • header - раздел шапки веб-страницы
  • nav - тег для создания меню
  • main - основной контент веб-страницы
  • section - задаёт раздел веб-страницы
  • footer - раздел подвала веб-страницы
  • div - универсальный контейнер
